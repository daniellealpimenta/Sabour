Política de Privacidade do Aplicativo Sabour

Data da última atualização: 4 de junho de 2025

Bem-vindo ao Sabour! A sua privacidade é importante para nós. Esta Política de Privacidade explica como o aplicativo Sabour lida com as informações para fornecer suas funcionalidades.

1. Informações que Não Coletamos Diretamente

O Sabour foi projetado para respeitar sua privacidade. Nós não coletamos, solicitamos, armazenamos ou transmitimos diretamente nenhuma informação pessoalmente identificável sua, como nome, endereço de e-mail, número de telefone, endereço físico, informações financeiras, dados de saúde ou localização precisa.

2. Dados Utilizados para a Funcionalidade do Aplicativo

Para oferecer a melhor experiência, o Sabour utiliza certos tipos de dados, principalmente para a funcionalidade do aplicativo:

  • Conteúdos de Jogos (Armazenados Localmente):
    • Dados: Seu progresso nos desafios culinários de cada cultura e sua lista de receitas favoritas.
    • Uso: Estes dados são salvos apenas no seu dispositivo para permitir que você continue seu progresso e acesse seus favoritos. Nós não temos acesso a esses dados.
    • Finalidade na App Store: Funcionalidade do app.
  • Histórico de Buscas (Interação com API):
    • Dados: Quando você busca uma receita, o termo de busca é enviado à API TheMealDB.com.
    • Uso: Utilizado pela API TheMealDB.com para retornar os resultados da sua busca. O Sabour não armazena seu histórico de buscas para outros fins.
    • Finalidade na App Store: Funcionalidade do app.
  • Identificador do Dispositivo (Endereço IP para Serviços de Terceiros):
    • Dados: Seu endereço IP é transmitido como parte da comunicação de rede padrão quando o Sabour se conecta à API TheMealDB.com para buscar receitas ou quando o player do YouTube carrega um vídeo.
    • Uso: Necessário para a comunicação de rede e para que esses serviços de terceiros funcionem. O uso do seu IP por esses serviços está sujeito às políticas de privacidade deles. O Sabour não usa seu IP para rastreá-lo ou vinculá-lo à sua identidade.
    • Finalidade na App Store: Funcionalidade do app.
  • Interações com o Produto (Armazenadas Localmente):
    • Dados: Informações como se você já viu as telas de introdução (onboarding) e suas interações gerais de navegação.
    • Uso: Para melhorar sua experiência e para o funcionamento correto do fluxo do aplicativo, salvando preferências localmente no seu dispositivo.
    • Finalidade na App Store: Funcionalidade do app.

3. Serviços de Terceiros

O Sabour interage com os seguintes serviços de terceiros:

  • TheMealDB.com API: Usada para buscar dados de receitas. A interação está sujeita aos termos e políticas da TheMealDB.
  • YouTube: Usado para exibir vídeos de receitas. A visualização está sujeita aos Termos de Serviço e Política de Privacidade do YouTube (Google).
  • DotLottie: Usado para animações visuais carregadas localmente, não transmitindo dados do usuário.

Recomendamos que você leia as políticas de privacidade desses terceiros para entender como eles lidam com seus dados.

4. Dados de Diagnóstico (Coletados pela Apple)

  • Dados de Falhas e Desempenho:
    • Dados: Se você optar por compartilhar com desenvolvedores, a Apple coleta dados de falhas (crash logs) e dados de desempenho (como tempo de inicialização, uso de energia) de forma anonimizada e agregada.
    • Uso: Usamos esses dados para identificar e corrigir bugs, entender a estabilidade do aplicativo e otimizar o desempenho.
    • Finalidade na App Store: Funcionalidade do app, Análise.

5. Privacidade Infantil

O Sabour não se destina ao uso por crianças menores de 13 anos. Não coletamos intencionalmente informações pessoalmente identificáveis de crianças.

6. Alterações a Esta Política de Privacidade

Podemos atualizar nossa Política de Privacidade periodicamente. Recomendamos que você revise esta página de tempos em tempos para quaisquer alterações.
